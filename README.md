# Sistem Sertifikat Digital Pelatihan

Aplikasi Next.js untuk penerbitan dan verifikasi sertifikat digital pelatihan **Junior Web Developer** pada bidang **Pengembangan Web**.

## Fitur Utama

- Portal admin, asesor, dan peserta
- Satu pelatihan tetap: Junior Web Developer
- Verifikasi publik tanpa login melalui endpoint terbatas
- Penyimpanan sertifikat digital terhubung ke blockchain dan IPFS

## Pengembangan

```bash
npm install
npm run dev
npm run lint
npm run build
```
